### Brief Introduction to NextJS

Head over to the NextJS [home page](https://nextjs.org/) to find out more. It does a wonderful job of introducing the framework.

Some things that Next JS do:

- Fullstack ReactJS framework for building website, with good SEO and additional features and optimizations, like a manufacturer
- Tooling automatically configured under the hood: bundling, compiling, dev server, ...
- Routing: file-system based routing
- Optimizations: Image, Fonts, Links, code splitting, ...
- Styling: Support for TailwindCSS
- Typescript
- Rendering: Client Side Rendering vs. Pre-rendering

### Zero Configuration

- Set up a Production Ready website with just a single command: create-next-app
- Little things done well with no configs needed: images and links
- Appreciation Vercel [products](https://vercel.com/) and mindsets: helping developers to focus on building app without spending time on config hell (üëã webpack, gulp, loaders)
- Vercel Hosting : Use Github to update project with just one button
- Auto CI-CD
- [Preview](https://vercel.com/features/previews) with comments: Faster iteration
- Serverless and the [Edge](https://vercel.com/features/infrastructure)
- [Databases](https://vercel.com/storage/kv)
- Monorepos: [Turborepo](https://turbo.build/repo)
- Build Tools and Bundling Tool: [TurboPack](https://turbo.build/pack)
